Making the web page

I wanted to create the web page with a simple and clear structure. I also wanted to try and ake the web page responsive, this turned out to be very challenging (and educational). I've uses some different semantic tags. For the main structure i have "header", "main" and "footer", I have also used article. I used div for making buttons, as text boxes and for making the header and footerI've used links with 3 different hover properties, and made an attempt at creating a sound hierarchy for my H1, H2, H3. To help my web page get responsive i created some rules for screens smaller than 624px, and 400px. I chose these values because this was when the boxes and heading started getting cramped.
